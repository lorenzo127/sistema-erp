Cómo conectar con Power BI / KNIME
  1. Descarga los archivos .csv usando los botones de arriba.
  2. Abre Power BI Desktop.
  3. Selecciona Obtener Datos > Texto/CSV.
  4. Los archivos ya vienen con codificación UTF-8 y separadores estándar.
  5. Tip Pro: Como los datos ya están limpios, puedes arrastrar directamente al canvas para crear gráficos.
